Sound Bloom Studio

KP Studio のモバイル向け音楽練習アプリです。

Sound Bloom Studio は、チューナー、メトロノーム、ドローン音、録音、ループ再生、再生速度変更、音声ファイルのインポートを備えた、日々の練習のためのアプリです。

Sound Bloom Studio is a mobile music practice app by KP Studio, with tuner, metronome, drone tones, recording, loop playback, playback speed control, and audio file import.
